# videos/ — sample videos for reproducing SOPBench experiments

All videos here are stored in **Git LFS**. After cloning the repo, you may need to run `git lfs pull` to download the actual MP4 files (otherwise you'll get small pointer files).

## Subdirectories

### `captaincook4d_samples/` (168 MB, 5 videos with audio)

5 first-person egocentric cooking videos from the [CaptainCook4D](https://github.com/CaptainCook4D) dataset. GoPro head-mounted camera, 360p, ~3–6 minutes each.

| ID | Recipe | Duration | Steps | Has errors |
|----|--------|----------|-------|-----------|
| 12_51 | Tomato Mozzarella Salad | 231s | 9 | Yes |
| 10_50 | Pinwheels | 239s | 13 | Yes |
| 10_6 | Pinwheels (correct) | 304s | 19 | No |
| 9_45 | Mug Cake | 315s | 11 | Yes |
| 1_34 | Microwave Egg Sandwich | 336s | 11 | Yes |

`annotations.json` here contains the step-level temporal annotations (start/end seconds, descriptions, error flags) extracted from CaptainCook4D's published annotation repo.

**Provenance:** Downloaded directly from CaptainCook4D's public Box CDN (URLs in their [downloader repo](https://github.com/CaptainCook4D/downloader)). License: Apache 2.0.

### `captaincook4d_samples_noaudio/` (133 MB, 5 videos)

The same 5 videos with the audio track stripped via `ffmpeg -an`. Used for the audio-off ablation in `docs/exp_log.md` (Experiment 10).

```bash
# Reproduce:
for f in videos/captaincook4d_samples/*.mp4; do
  ffmpeg -i "$f" -c:v copy -an "videos/captaincook4d_samples_noaudio/$(basename $f)"
done
```

### `probe/` (8.6 MB, synthetic test videos)

Synthetic 30-second videos with random 4-character codes displayed on black background, used to verify Gemini's actual frame sampling behavior at different fps (Experiment 6 in `docs/exp_log.md`).

- `probe_30s.mp4` — codes change every 0.25s
- `probe_1s_codes.mp4` — codes last 1.0s each
- `probe_0.5s_codes.mp4` — codes last 0.5s each
- `probe_0.25s_codes.mp4` — codes last 0.25s each
- `ground_truth.json`, `probe_v3_balanced.json` — the known answer keys + measured Gemini accuracy

Generated by `sopbench/archive/probe_video.py` (v1) and tested by `sopbench/ablations/probe_balanced.py` (v3, the bug-fixed version).
